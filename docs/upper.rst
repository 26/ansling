upper
=====

`upper converts the given string to uppercase.

Description
-----------

.. code-block:: text

   upper `string` value : `string`

`upper` converts `value` to uppercase and returns the result.

Parameters
----------

value
*****

The string that must be converted to uppercase.

Examples
--------

1. Basic usage
**********************

.. code-block:: text

   upper "FooBar"

   Result:

   FOOBAR

.. code-block:: text

   upper "foobar"

   Result:

   FOOBAR

.. code-block:: text

   upper "FOOBAR"

   Result:

   FOOBAR
