flip
====

`flip` reverses the order of the elements of the given array and returns the result.

Description
-----------

.. code-block:: text

   flip `mixed array` array : `string`

`flip` reverses the order of the elements of `array` and returns the result as an array.

Parameters
----------

array
******
The input array.

Examples
--------

1. Basic usage
**********************

.. code-block:: text

   flip split "," "foo,bar"

   Result:

   ["bar", "foo"]
