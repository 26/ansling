sort
====

`sort` sorts the given array alphabetically from A to Z.

Description
-----------

.. code-block:: text

   sort `string array` array : `string array`

This commands sorts `array` in alphabetical order from A to Z.

Parameters
----------

array
*****
The input array.

Examples
--------

1. Basic usage
**********************

.. code-block:: text

   sort split "," "b,d,e,c,a"

   Result:

   ["a", "b", "c", "d", "e"]
