lower
=====

`lower` converts the given string to lowercase.

Description
-----------

.. code-block:: text

   lower `string` value : `string`

`lower` converts `value` to lowercase and returns the result.

Parameters
----------

value
*****

The string that must be converted to lowercase.

Examples
--------

1. Basic usage
**********************

.. code-block:: text

   lower "FooBar"

   Result:

   foobar

.. code-block:: text

   lower "FOOBAR"

   Result:

   foobar

.. code-block:: text

   lower "foobar"

   Result:

   foobar
