length
======

`length` returns the number of characters, or length, of the given string.

Description
-----------

.. code-block:: text

   length `string` value : `integer`

`length` returns the length of `value` as an integer.

Parameters
----------

value
*****

The string of which the length must be returned.

Examples
--------

1. Basic usage
**********************

.. code-block:: text

   length "foobar"

   Result:

   6
